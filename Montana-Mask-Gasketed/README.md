
Printer settings

Nozzle Size: 0.4 mm\
Layer Height: 0.3 mm\
Infill: 15%  
Walls: 2\
Top/Bottom Layers: 3  

Temp is material dependent, we usually print with 200C nozzle and 50C bed.

Make sure you print both the mask and the filter.
